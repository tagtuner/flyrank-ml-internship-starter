# Portfolio Identity Kit: OmniTech

This document details the visual style parameters and style guide for the portfolio to ensure visual consistency across all pages.

---

## 1. Brand Tokens

*   **Name**: OmniTech
*   **Logo Symbol**: Monogram **OT**
*   **Mood**: A clean, quiet, engineering-focused framework that keeps live security logs and rule metrics as the primary focal points.

---

## 2. Typography

*   **Heading Font**: **Outfit** (Google Fonts) - Crisp, geometric, modern sans-serif. Used for titles, page headers, and card titles.
*   **Body Font**: **Inter** (Google Fonts) - Clean, highly legible, optimized for interface text. Used for descriptions, logs, tables, and paragraphs.
*   **Code / Monospace**: **Geist Mono** or system monospace. Used for firewall rule dumps, routing logs, and code configurations.

---

## 3. Color Palette

*   **Background (Near-White)**: `#fcfbfa` (Alabaster Off-White) - Warm, premium background tone.
*   **Text (Near-Black)**: `#1a1a19` (Charcoal) - Soft on the eyes, avoids harsh solid black contrast.
*   **Primary / Accent**: `#cc785c` (Terracotta) - Muted, warm red-orange, used sparingly for links, active badges, and CTAs.
*   **Borders / Dividers**: `#e6e4df` (Soft Gray) - Thin divider line separators.

---

## 4. Reusable Style Note
> "Fonts: Outfit (headings) & Inter (body). Colors: #fcfbfa (bg), #1a1a19 (text), #cc785c (terracotta). Mood: A clean, quiet, engineering-focused framework that keeps live security logs and rule metrics as the primary focal points."
